DOP/1

Почему цвет #BruceWillis является цветом #B0E000 ?

  1. Заменяем все неправильные шестнадцатеричные символы нулями 0, BruceWillis превращается в B00CE000000
  2. “Добиваем” число нулями до количества знаков, делимого на 3 (11 -> 12), получаем B00C E000 0000
  3. Разбиваем на три группы, каждая из которых отвечает за одну компоненту RGB: RGB (B00C, E000, 0000)
  4. Обрезаем каждый из аргументов справа, оставляя 2 символа, и получаем наш результат RGB (B0, E0, 00) = #B0E000

Почему цвет #CANDY является цветом #CA0000 ?

  1. Заменяем все неправильные шестнадцатеричные символы нулями 0, CANDY превращается в CA0D0000000
  2. “Добиваем” число нулями до количества знаков, делимого на 3 (11 -> 12), получаем CA0D 0000 0000
  3. Разбиваем на три группы, каждая из которых отвечает за одну компоненту RGB: RGB (CA0D, 0000, 0000)
  4. Обрезаем каждый из аргументов справа, оставляя 2 символа, и получаем наш результат RGB (CA, 00, 00) = #CA0000

DOP/2

Палитра состоит из 216 безопасных веб-цветов. Под каждым цветом указаны два значения — RGB, для создания цвета в графическом редакторе, и его шестнадцатеричных HEX-код, для обозначения цвета в HTML.


Актуальность WEB SAFE палитры:

  • Безопасные цвета всегда неизменны при переходе от одного браузера к другому, от одного монитора к другому, или от одной платформы к другой, с их различными разрешениями и возможностями цветопередачи.
  • Если ваш веб-сайт имеет какой-либо цветной фон, то лучше всего для этой цели использовать именно Web Safe цвета. Таким образом, это гарантирует вам, что цвет не будет смущать пользователя при отображении на разных компьютерных платформах.
  • Даже старые компьютеры будут представлять ресурс точно и без колебаний и различий в цветах.
  • Если Вы создаете картинки, то следует обратиться к WEB SAFE палитре, так как это позволит получать более чистое изображение, отображаемое и для вас, и для пользователя одинаково.
